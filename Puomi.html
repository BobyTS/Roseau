    There are few approaches to accessing the Puomi client, all requiring a 32-bit browser.

    The latest Shockwave works better in Mozilla-based (Firefox-like) browsers than in Internet Explorer.
    Firefox 52 has most NPAPI plugins disabled including Shockwave. There is a preference setting in about:config to re-enable, but finding an older version (36-38, even older will work too) or alternative (such as Pale Moon (32-bit) / portable version) will work in the long run. Latest version of Shockwave is at Adobe's site.

    If you're using a very old Firefox release or Internet Explorer, you can try an older version of Shockwave. (10.1.3)

    Restarting the system after Shockwave install is recommended.
    If reinstalling Shockwave, running Shockwave Uninstaller between setups is a good idea.

    If you are unable to use dice or drop-down boxes like Say/Shout/Whisper, try loading the client at puomi.tunk.org/p/5/ and keeping it in the background for a minute.
